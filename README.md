
# Analog Clock

### This website is deployed [here.](https://rajatkhatri7.github.io/CLOCK/)


### *Features*:
* User-friendly
* Displays current Time
* Neomorphic Interface 
* Added Themes: Dark Mode <-> Light Mode


## *Technologies used*

- HTML5
- CSS3 (with Flexbox)
- Vanilla Javascript
